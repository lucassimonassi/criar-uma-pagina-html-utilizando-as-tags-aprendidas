  1. <h1> até <h6>

    São tags utilizadas em HTML para definir os cabeçalhos ou títulos de uma página.

    <h1> é o cabeçalho mais importante, sendo o principal da página.

    <h2> é utilizado para títulos secundários e está abaixo do título '<h1>'. Pode se ter vários '<h2>' e eles são frequentemente usados para dividir os conteúdos em sessões.

    <h3> a <h6> são usados para títulos hierarquicos cada vez menores. O '<h3>' é mais importante que o '<h4>', que é mais importante que o '<h5>' e '<h6>'.

  15. <font>

    A tag <font> manipula o tamanho, cor e tipo da fonte.
    Exemplo: <font size="4" color="red" face="Arial">
    Resultado: Este é um texto vermelho com tamanho 4 e fonte Arial

  17. <abbr>

    A tag <abbr> é usada para definir uma abreviação ou acrônimo.
